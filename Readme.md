# Believers Expense Tracker

A smart expense tracking application built with 
**React JS** that allows users to manage their daily expenses, filter and analyze them over time, and visualize spending using charts. Data is persisted using **localStorage** to ensure entries remain across sessions.

---

## 🚀 Features

- **Authentication**
  - Login and Sign Up pages
  - Authorization for secure access
  - Logout functionality

- **Dashboard**
  - Overview of income and expenses
  - Quick access to filters and charts

- **Expense Entry**
  - Input fields for amount, category, date, and note
  - Predefined categories: Food, Travel, Bills, Others
  - Display expenses grouped by recent entries

- **Category-wise Tracking**
  - Categorize each expense entry
  - Filter by selected category
  - View total spent per category

- **Filters and Sorting**
  - Filter by date, category, or amount range
  - Sort by most recent or highest amount

- **Charts & Visualization**
  - Pie chart or bar graph showing category-wise expense distribution
  - Dynamic updates as expenses are added or removed

- **User Controls**
  - Add, view, and delete expense entries
  - Search expenses by date, amount, or category
  - View total and category-wise spending breakdown

---

## 🛠 Tech Stack

- **React JS** – Frontend framework
- **TailwindCSS** – Styling
- **localStorage** – Data persistence
- **Chart.js / Recharts** – Data visualization
